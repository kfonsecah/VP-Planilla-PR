# Phase 45, Plan 02: Tab "Resumen" (Widgets Consolidados)

## Objetivo
Implementar la pestaña principal "Resumen" que muestra summaries (resúmenes rápidos) de las áreas críticas del empleado, cumpliendo la decisión de alta densidad pero con diseño drill-down.

## Tareas

1. **Tarjeta de Información Base**:
   - Info personal: Cedula, Email, Teléfono, Fecha de ingreso.
   - Puesto y departamento/sucursal.
   - Botón de "Editar Info" que invocará el modal existente.

2. **Widget de Tiempo y Marcas**:
   - Mostrar estado de reloj (Aliases asignados actuales).
   - Mostrar contador de Vacaciones Disponibles.
   - Mostrar últimas 3 marcas de esta quincena.

3. **Widget de Compensación**:
   - Salario Base actual.
   - Mostrar el Neto del último pago de planilla.

4. **Integración en `page.tsx`**:
   - Renderizar este contenido cuando el tab seleccionado sea "Resumen".

## Verificación
- El Tab "Resumen" pinta tarjetas distribuidas en CSS Grid / Flexbox.
- Se muestran solo top-N registros (marcas, vacaciones) y no tablas gigantes.
