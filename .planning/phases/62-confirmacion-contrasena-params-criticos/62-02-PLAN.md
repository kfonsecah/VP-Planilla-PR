# Phase 62-02: Password Confirmation Modal & Integration - Plan

## Wave 2: Frontend UI
**Goal:** Create a reusable password confirmation modal and provide a hook for integration.

### Task 4: PasswordConfirmModal Component
- **File:** `src/frontend/src/components/PasswordConfirmModal.tsx`
- **Action:** Create component using Tailwind 4 + Zinc-950/Zinc-900 style.
- **Features:**
  - Password input with "Show password" toggle (eye icon).
  - Modal must NOT close automatically if `error` is present (persistence on 403).
- **Style guide:** Reference `ChangePasswordModal.tsx`.
- **Props:**
  - `isOpen`: boolean
  - `paramName`: string
  - `onConfirm`: (password: string) => void
  - `onCancel`: () => void
  - `isLoading?`: boolean
  - `error?`: string
- **Acceptance criteria:**
  - Component matches design system.
  - Password input is hidden by default.
  - Submit button shows loading spinner.
  - Error message displays if password validation fails; modal remains open.

### Task 5: usePasswordConfirmation Hook
- **File:** `src/frontend/src/hooks/usePasswordConfirmation.ts`
- **Action:** Create a hook to manage modal state and action execution.
- **Acceptance criteria:**
  - Hook provides a simple API to wrap any function with password confirmation.
  - Correctly handles the flow: open modal -> wait for password -> call action -> handle success/failure.

### Task 6: Preparation for Panel Admin
- **File:** `src/frontend/src/services/legalParamService.ts`
- **Action:** 
  - Ensure `patchParam` and `upsertParam` accept the `confirmationPassword` optional field.
  - Update `LegalParam` interface to include `isCritical: boolean`.
- **Acceptance criteria:**
  - Type definitions updated.
  - HTTP requests include the field when provided.
  - Frontend components can read `isCritical` from the parameter object.

## Verification Criteria (Nyquist D8)
1. **Frontend UI:** Component renders correctly and matches the visual theme.
2. **Integration:** A mock test demonstrates the modal appearing and blocking an action until correct password entry.
