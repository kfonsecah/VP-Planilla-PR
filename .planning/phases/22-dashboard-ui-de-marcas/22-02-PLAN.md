**1. Create `src/frontend/src/hooks/useClockLogs.ts`:**

```typescript
import { useState, useEffect, useCallback } from 'react';
import { ClockLogsService, ClockLogPaginated, ClockLogStats, ImportSession } from '@/services/clockLogsService';
import { getEmployees } from '@/services/employeeService';
```

Hook signature:
```typescript
export function useClockLogs()
```

State:
- `stats: ClockLogStats | null` (initially null, loading state)
- `logs: ClockLogPaginated[]` (initially [])
- `totalLogs: number` (initially 0)
- `page: number` (initially 1)
- `pageSize: number` (initially 20, constant)
- `importSessions: ImportSession[]` (initially [])
- `isLoading: boolean` (initially true)
- `isStatsLoading: boolean` (initially true)
- `error: string | null`
- `filters`: object with `initDate: string` (default: first day of current month YYYY-MM-DD), `endDate: string` (default: today YYYY-MM-DD), `status: string[]` (default: []), `employee_id: number | undefined`
- `employees: Array<{ id: number; name: string }>` (for autocomplete, loaded once)

Date defaults: compute using `new Date()` — initDate = `YYYY-MM-01`, endDate = `YYYY-MM-DD` of today.

Functions (all wrapped in `useCallback`):
- `fetchStats()`: calls `ClockLogsService.getStats(filters.initDate, filters.endDate)`, sets stats
- `fetchLogs()`: calls `ClockLogsService.getClockLogsPaginated({ ...filters, page, pageSize })`, sets logs + totalLogs
- `fetchImportSessions()`: calls `ClockLogsService.getImportSessions(5)`, sets importSessions
- `fetchAll()`: calls all three above, sets isLoading
- `setPage(n: number)`: updates page, triggers fetchLogs
- `setFilters(newFilters: Partial<typeof filters>)`: merges filters, resets page to 1
- `applyDatePreset(preset: 'today' | 'last7days' | 'thisMonth')`: computes initDate/endDate and calls setFilters
  - today: both dates = today
  - last7days: initDate = today - 6 days, endDate = today
  - thisMonth: initDate = first of month, endDate = today

Effects:
- `useEffect` on mount: fetch employees list via `getEmployees()`, map to `{ id, name: first_name + ' ' + last_name }` or whatever shape Employee has
- `useEffect` on `[filters.initDate, filters.endDate, filters.status, filters.employee_id]`: call `fetchAll()` (resets page to 1)
- `useEffect` on `[page]`: call `fetchLogs()` only (stats/sessions don't change with page)

Return shape:
```typescript
return {
  stats, logs, totalLogs, page, pageSize,
  importSessions, isLoading, isStatsLoading, error,
  filters, employees,
  setPage, setFilters, applyDatePreset
};
```
