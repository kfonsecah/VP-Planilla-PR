## Interaction Contracts

### Modal Triggers
- **Day card**: Edit icon → opens "Editar marca" modal for that specific day
- **Employee header**: "Agregar marca" button → opens "Agregar marca faltante" modal with employee pre-filled and locked

### Form Validation
- Justification field: minimum 10 characters (MARCAS-02 requirement)
- Live character count display inline (CONTEXT.md D-04)
- Error on submit if validation fails

### Preview Before Confirm
- Before final submission, show text: "Se agregará una marca de entrada para Juan Pérez el 15/04/2026 a las 08:00"
- Applies to all three actions (add/edit/void)

### Audit Timeline Display
- Badge with count on day row (e.g., "(3 cambios)")
- Hover tooltip shows summary
- Click expands full timeline within day card

### Destructive Action Confirmation (UX-02)
- Modal with text input required
- User must type "ANULAR" to confirm void action
- Matches CONFIRMAR requirement from UX-02
