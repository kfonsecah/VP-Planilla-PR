## Component Inventory

### 1. LegalParamAlertBanner (new)

Location: `src/frontend/src/components/LegalParamAlertBanner.tsx`

**Trigger:** Renders above the stats grid in `main/page.tsx` when `GET /notifications?type=LEGAL_PARAM_CHANGE&unacknowledged=true` returns one or more results (admin or payroll_manager roles only).

**Layout:**
```
[ShieldExclamationIcon 20px] [Title: "ParГЎmetro legal modificado"] [collapse chevron] [Г—]
                              [Expanded: param name, old в†’ new, who, when]
                              [If multiple: "N alertas sin revisar вЂ” ver todas"]
                              [CTA: "Marcar como revisado" вЂ” admin only]
```

**States:**
- Single alert, expanded by default: shows full message + CTA
- Multiple alerts: shows first one expanded, secondary ones collapsed with count badge (`+N mГЎs`)
- All acknowledged: component unmounts (AnimatePresence exit)
- Loading: skeleton pulse (same pattern as `NotificationPanel.tsx` lines 98вЂ“110)

**Behavior:**
- Collapse/expand: chevron toggle, no page navigation
- "Marcar como revisado": calls `PATCH /notifications/:id/acknowledge`, then refetches; button shows spinner during request. Button style follows `configuracion/empresa/page.tsx` primary button pattern: `bg-green-600 hover:bg-green-700 text-white rounded-xl font-semibold px-6 py-2.5 shadow-md shadow-green-600/20` вЂ” disabled state: `bg-zinc-200 dark:bg-zinc-800 text-zinc-400 cursor-not-allowed shadow-none`.
- Dismiss (Г—): hides locally for the session only вЂ” does NOT mark as acknowledged
- Admin-only: `acknowledged_by` button hidden for `payroll_manager` role

**Animation:** `AnimatePresence` + `motion.div` with `backdropVariants` / `modalVariants` (same pattern as `NotificationPanel.tsx` lines 22вЂ“31). Entry: `opacity 0в†’1, y -8в†’0` in 150ms. Exit: `opacity 1в†’0, y 0в†’-8` in 100ms.

### 3. NotificationPanel вЂ” LEGAL_PARAM_CHANGE differentiation (extend existing)

Location: `src/frontend/src/components/ui/NotificationPanel.tsx`

**Change:** When `notification.notification_type === 'LEGAL_PARAM_CHANGE'`, render a `ShieldExclamationIcon` (20px, `text-red-500`) instead of the generic blue dot. The panel row background uses `bg-red-50/50 dark:bg-red-900/10` instead of `bg-[#FFF9E6]`.

**Bell badge in Header:** When any unacknowledged `LEGAL_PARAM_CHANGE` exists, show a second indicator вЂ” a 6px solid red dot at bottom-right of the bell icon (in addition to the unread count badge at top-right). This signals legal urgency independently of general unread count.

## Interaction States

### LegalParamAlertBanner

| State | Visual |
|-------|--------|
| Loading (initial fetch) | 3-row skeleton pulse вЂ” matches NotificationPanel skeleton |
