# Plan 48-02: Limpieza de código muerto, Excel de prueba y carpetas vacías

## Objetivo

Eliminar código muerto del frontend (utilidades hardcodeadas reemplazadas por el backend), archivos Excel de prueba sueltos en el repositorio, reportes HTML ad-hoc, bitácoras PDF obsoletas, y scripts de generación que ya no tienen uso activo.

---

## Inventario de archivos a eliminar

### Código muerto — frontend/utils/holidays.ts

| Archivo | Motivo |
|---|---|
| `src/frontend/src/utils/holidays.ts` | Este archivo contiene la lógica hardcodeada de feriados CR (`getCostaRicaHolidays`). Fue reemplazado completamente por el motor de feriados del backend implementado en Phase 44 (`companyHolidayService.ts`). El frontend ahora usa `useHolidays` que llama al API. Este archivo **no tiene ningún import** activo en el codebase según grep. |

**PRECAUCIÓN**: Antes de eliminar, verificar con grep que no hay imports activos de `holidays.ts` o `getCostaRicaHolidays` en el frontend. Si hay algún import residual, debe actualizarse para usar el hook `useHolidays` en su lugar.

### Excel de prueba suelto

| Archivo | Motivo |
|---|---|
| `src/frontend/marcas_prueba.xlsx` | Archivo Excel de prueba colocado en la raíz del proyecto frontend. Los archivos de muestra viven en `docs/samples/` y `src/frontend/public/samples/`. Este archivo no tiene propósito en el repo. |

### Reportes HTML y bitácoras PDF obsoletos en docs/

| Archivo | Motivo |
|---|---|
| `docs/test-report.html` | Reporte HTML generado de tests. Artefacto de ejecución. |
| `docs/reporte-profesor.html` | Reporte HTML ad-hoc para presentación académica. No es documentación viva del sistema. |
| `docs/VP_Planilla_Bitacora_Pruebas_v2.pdf` | Bitácora de pruebas v2 — obsoleta. Reemplazada por v3 y v4. |
| `docs/VP_Planilla_Bitacora_Pruebas_v3.pdf` | Bitácora v3 — potencialmente obsoleta. Mantener solo la más reciente (v4). |

> **NOTA**: Conservar `docs/VP_Planilla_Bitacora_Pruebas_v4.pdf` como la bitácora vigente.

### Scripts de generación en docs/scripts/

| Archivo | Motivo |
|---|---|
| `docs/scripts/generate-attendance-sample-xlsx.cjs` | Script para generar samples de Excel. Duplicado — existe versión activa en `src/frontend/scripts/`. |
