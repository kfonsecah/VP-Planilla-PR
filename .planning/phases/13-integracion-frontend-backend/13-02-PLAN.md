       - Zod validation errors: backend may return { success: false, errors: [{ field: "name", message: "..." }] } — extract and format these

       ```
       const msg = data?.message || data?.error || JSON.stringify(data);
       ```
       Enhance to also check: data?.errors (array), data?.validationErrors, data?.details

    Do NOT change the token refresh logic or authentication flow — those are working correctly.
