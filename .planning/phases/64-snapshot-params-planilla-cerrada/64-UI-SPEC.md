## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |

- Table cell padding: `px-4 py-4` (md + md)

## Layout & Interactive Elements

### Section Container
- **Visibility:** Only displayed when payroll status is `APROBADA` or `PAGADA`
- **Hidden:** When status is `BORRADOR` or `CANCELADA`
- **Border:** `border border-zinc-200 dark:border-zinc-800 rounded-lg`
- **Padding:** `p-6`
- **Margin Top:** `mt-8` (major gap from previous section)

### Collapse Toggle
- **Type:** Click the entire header row to expand/collapse
- **Icon:** Lucide `ChevronDown` (rotates 180° when open)
- **Animation:** `transition-transform duration-200`
- **Cursor:** `cursor-pointer`

### Table Structure
- **Container:** `w-full overflow-x-auto` (responsive scrolling on mobile)
- **Header Row:** `bg-zinc-100 dark:bg-zinc-800 border-b border-zinc-300 dark:border-zinc-700`
- **Header Cell:** `px-4 py-4 text-left text-xs text-zinc-700 dark:text-zinc-300 uppercase tracking-wider`
- **Body Rows:** Alternating `bg-white dark:bg-zinc-900/50` and `bg-zinc-50 dark:bg-zinc-900`
- **Body Cell:** `px-4 py-4 text-sm text-theme-primary dark:text-zinc-100 border-b border-zinc-200 dark:border-zinc-800`
- **Parameter Name (Column 1):** `text-theme-primary dark:text-white`
- **Value (Column 2):** `font-bold text-[#4A5D3A] dark:text-white`
- **Date (Column 3):** `text-xs text-zinc-600 dark:text-zinc-400`
- **Decree (Column 4):** Badge format: `inline-block bg-blue-100 dark:bg-blue-900/30 text-blue-700 dark:text-blue-400 px-2 py-1 rounded text-xs`

### Category Grouping
- **Sub-headers within table:** Striped background `bg-zinc-200 dark:bg-zinc-700/50` across full width
- **Category text:** Bold, left-aligned in first column, no padding difference
- **Indentation:** Child rows indented 1 level (add left padding)

### Bottom Notice (if historical)
- **Text:** "Estos valores fueron vigentes en la fecha de cierre de planilla. Parámetros actuales pueden diferir."
- **Style:** `text-xs text-zinc-600 dark:text-zinc-400 italic mt-4 pt-4 border-t border-zinc-200 dark:border-zinc-700`

## Accessibility (A11y)

- **Focus Management:** Collapse toggle has `focus:outline-none focus:ring-2 focus:ring-[#4A5D3A] focus:ring-offset-2`
- **ARIA:**
  - Toggle button: `aria-expanded={isOpen}`
  - Table: `role="grid"` or semantic `<table>` with `<thead>` and `<tbody>`
  - Category rows: `role="rowgroup"` or visual grouping via CSS
- **Semantics:** Use `<table>` + `<thead>` + `<tbody>` for proper screen reader experience
- **Color Alone:** Badge decrees use icon (document icon) + color to avoid color-only differentiation
- **Text Contrast:** All text meets WCAG AA (4.5:1 minimum)

## Notes for Executor

1. **Snapshot Data Shape:** Backend returns array of `{ id, payroll_id, param_key, param_value, param_valid_from, source_decree, captured_at }`
2. **Grouping Logic:** Categorize params by prefix (e.g., `OT_`, `CCSS_`, `ENTERPRISE_`, `GLOBAL_`)
3. **Enterprise Params:** Special handling for `ENTERPRISE_MINUTE_ROUNDING_POLICY`, `ENTERPRISE_ORDINARY_SHIFT_TYPE`, `ENTERPRISE_IS_COMMERCIAL_ACTIVITY` — these have string enum values, display human-readable labels
4. **Date Formatting:** Use `toLocaleDateString('es-CR', { year: 'numeric', month: 'long', day: 'numeric' })` for Spanish Costa Rican locale
5. **Decree Badge:** Only show if `source_decree` is not null/empty
6. **Mobile Table:** Consider wrapping decree and date columns into a secondary row or tooltip for mobile breakpoint
