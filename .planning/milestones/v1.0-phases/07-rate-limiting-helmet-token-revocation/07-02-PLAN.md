# Plan 07-02: Rate Limiting en Login

## Requisitos
- REQ 7.1: Rate limiter en `POST /api/login` (máx 10 req / 15 min / IP)
- REQ 7.2: 11° intento retorna 429 Too Many Requests

## Tareas

### Tarea 1: Agregar rate limiter al login
<read_first>
src/backend/src/routes/AuthRoute.ts (líneas 1-70)
</read_first>

<action>
Agregar imports al inicio del archivo (después de línea 3):
```typescript
import rateLimit from 'express-rate-limit';
```

Agregar el limiter después de los imports (antes del primer router.):
```typescript
const loginLimiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutes
  max: 10, // 10 requests per window per IP
  message: {
    success: false,
    error: 'Demasiados intentos de login. Intente de nuevo en 15 minutos.'
  },
  standardHeaders: true,
  legacyHeaders: false,
});
```

Modificar la ruta de login para usar el limiter (buscar `router.post("/login"` y agregar el limiter):
```typescript
router.post("/login", loginLimiter, asyncHandler(AuthController.login));
```
</action>
