# Plan 05-01 — Eliminar Ruta Deprecated POST /nominee/calculate

**Phase:** 05 — Funcionalidad de Negocio Faltante

## Problem

`POST /api/nominee/calculate` es una ruta legacy marcada como `@deprecated` en NomineeService (línea 105). Internamente hardcodea `employeeSalary = 1000` (línea 149), produciendo resultados incorrectos. Fue reemplazada por `POST /api/nominee/calculate-payroll`.

## Locations to Modify

| File | Lines | Action |
|------|-------|--------|
| `src/backend/src/routes/NomineeRoute.ts` | 78–97 | Remove route + swagger docs |
| `src/backend/src/controller/NomineeController.ts` | 56–76 | Remove method |
| `src/backend/src/service/NomineeService.ts` | 102–158 | Remove method |

## Steps

1. **NomineeRoute.ts** — Delete lines 78–97 (swagger comment block + route registration)
2. **NomineeController.ts** — Delete lines 56–76 (`calculateNominee` method)
3. **NomineeService.ts** — Delete lines 102–158 (`calculateNominee` method + deprecation comment)
4. Run `npx tsc --noEmit` in `src/backend/`
5. Verify route returns 404: start server, `curl -X POST http://localhost:3001/api/nominee/calculate`
