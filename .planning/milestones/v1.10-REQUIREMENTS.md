## 2. Security (SEC)

- [x] **SEC-01**: Implementar middleware `hpp` en Express para protección contra HTTP Parameter Pollution.
  - **Outcome:** `hpp` package installed, `preventParameterPollution` exported from `queryNormalizer.ts`, applied globally. ✅ Phase 75.
- [x] **SEC-02**: Implementar un middleware global de normalización de `req.query` para mitigar riesgos de Express 5 y HPP, convirtiendo parámetros múltiples en valores únicos (last-value-wins). La refactorización con Zod se difiere a deuda técnica.
  - **Outcome:** `normalizeQuery` uses `Object.defineProperty` shadow pattern. Applied as first element of `querySecurityMiddleware`. ✅ Phase 75.
