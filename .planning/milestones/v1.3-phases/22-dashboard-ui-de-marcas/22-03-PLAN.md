**Component typed as `React.FC<ClockLogDetailModalProps>`.**

**Internal state:**
- `auditLogs: Array<{ id: number; action: string; details: string; created_at: string; user_name?: string }>` (initially [])
- `isLoadingAudit: boolean` (initially false)
- `justification: string` (initially '')

- `showCorrectionForm: boolean` (initially false)

**Animation constants** (inside component, same as EditEmployeeModal):
```typescript
const backdropVariants = { hidden: { opacity: 0 }, visible: { opacity: 1 } };
const modalVariants = {
  hidden: { scale: 0.9, opacity: 0, y: 30 },
  visible: { scale: 1, opacity: 1, y: 0, transition: { type: 'spring' as const, damping: 20, stiffness: 250 } },
  exit: { scale: 0.9, opacity: 0, y: 30, transition: { duration: 0.2 } }
};
```

**useEffect on `[isOpen, log]`:**
- When `isOpen && log`: fetch audit logs via `ClockLogsService.getAuditLogsForClockLog(log.id)`, set auditLogs. Reset justification to '', showCorrectionForm to false.
- Focus modal ref.

**Layout (compact modal, max-w-2xl):**

Use AnimatePresence + MotionDiv backdrop + modal pattern exactly as EditEmployeeModal.

Modal content sections:

**Section 1: Log Details** (grid, 2 cols)
- Empleado: `log.employee_name`
- ID Empleado: `log.employee_id`
- Timestamp: formatted date/time
- Tipo: `log.log_type` (IN/OUT)
- Status: `<ClockLogStatusBadge status={log.status} />`
- Source: prettified source name
- Observaciones: `log.remarks || 'Ninguna'`
- Session ID: `log.import_session_id || 'N/A'`

**Section 2: Audit History**
- Heading: "Historial de Auditoria"
- If `isLoadingAudit`: "Cargando historial..."
- If `auditLogs.length === 0` and not loading: "Sin historial de cambios"
- Otherwise: list of audit entries, each showing:
  - Action (bold), details, date formatted, user if available
  - Use a compact vertical list with `border-l-2 border-zinc-300 pl-3 ml-2` for timeline feel

**Section 3: Correction Actions** (only show if `log.status !== 'corrected'`)
- If `!showCorrectionForm`: two buttons:
  - "Marcar como Corregido" (primary blue button) — sets `showCorrectionForm = true`
  - "Descartar" (red outline button) — sets `showCorrectionForm = true` with a discard flag
- If `showCorrectionForm`: show justification textarea + confirm/cancel buttons
  - `<textarea>` for justification, placeholder "Justificacion de la correccion...", required, min 5 chars
  - "Confirmar" button (calls handleCorrection) and "Cancelar" button (hides form)
  - `handleCorrection` async function:
    1. Validate justification length >= 5, else toast.error
    2. Set isSubmitting true
    3. Call `ClockLogsService.updateClockLogStatus(log.id, 'corrected', justification)`
    4. On success: toast.success("Marca actualizada correctamente"), call `onCorrected()`, call `onClose()`
    5. On error: toast.error with error message
    6. Set isSubmitting false

**Section 4: Footer**
- "Cerrar" button that calls `onClose()`

Use `modalRef` with `useRef<HTMLDivElement>(null)` for the modal container, same pattern as EditEmployeeModal.
