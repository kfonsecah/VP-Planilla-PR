El problema: `new Date("2026-04-06")` crea medianoche UTC, pero los timestamps en DB están en hora local Costa Rica (UTC-6).

CORRECCIÓN REQUERIDA:

1. Crear función helper para parsear fechas como hora local:
```typescript
function parseLocalDate(dateStr: string): Date | undefined {
  if (!dateStr) return undefined;
  const [year, month, day] = dateStr.split('-').map(Number);
  // Crea medianoche local (Costa Rica UTC-6)
  return new Date(year, month - 1, day);
}

function parseLocalDateEnd(dateStr: string): Date | undefined {
  if (!dateStr) return undefined;
  const [year, month, day] = dateStr.split('-').map(Number);
  // Fin del día local: 23:59:59.999
  return new Date(year, month - 1, day, 23, 59, 59, 999);
}
```

2. Reemplazar en el método getClockLogsPaginated (línea ~589):
```typescript
// ANTES (INCORRECTO):

// DESPUÉS (CORRECTO):
const initDate = req.query.initDate ? parseLocalDate(req.query.initDate as string) : undefined;
const endDate = req.query.endDate ? parseLocalDateEnd(req.query.endDate as string) : undefined;
```
