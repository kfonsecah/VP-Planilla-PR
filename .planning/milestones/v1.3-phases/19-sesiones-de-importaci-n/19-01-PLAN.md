       ```prisma
       model vpg_clock_import_sessions {
         import_sessions_id          Int              @id @default(autoincrement())
         import_sessions_started_at  DateTime         @default(now()) @db.Timestamp(6)
         import_sessions_completed_at DateTime?       @db.Timestamp(6)
         import_sessions_source      ClockLogSource
         import_sessions_status      String           @default("pending") @db.VarChar(20)
         import_sessions_total_records Int            @default(0)
         import_sessions_created_count Int            @default(0)
         import_sessions_skipped_count Int            @default(0)
         import_sessions_anomaly_count Int            @default(0)
         import_sessions_created_by  Int
         vpg_clock_logs              vpg_clock_logs[]
         vpg_users                   vpg_users        @relation(fields: [import_sessions_created_by], references: [user_id], onDelete: NoAction, onUpdate: NoAction, map: "fk_vpg_clock_import_sessions_users_25")

         @@index([import_sessions_started_at], map: "idx_vpg_clock_import_sessions_started_at")
         @@index([import_sessions_status], map: "idx_vpg_clock_import_sessions_status")
         @@index([import_sessions_source], map: "idx_vpg_clock_import_sessions_source")
         @@index([import_sessions_created_by], map: "idx_vpg_clock_import_sessions_created_by")
       }
       ```
       - Status values: pending, running, completed, failed (VARCHAR, not enum — Phase 20 may add more statuses)
       - Source reuses existing ClockLogSource enum
       - FK to vpg_users via import_sessions_created_by

    2. Add clock_logs_import_session_id field to vpg_clock_logs model:
       ```prisma
       clock_logs_import_session_id Int?
       vpg_clock_import_sessions    vpg_clock_import_sessions? @relation(fields: [clock_logs_import_session_id], references: [import_sessions_id], onDelete: SetNull, onUpdate: NoAction, map: "fk_vpg_clock_logs_import_sessions_26")
       ```
       - Nullable because manual entries (source: manual) may not have a session (per 19-CONTEXT.md decisions)
       - onDelete: SetNull — if session is deleted, logs remain but lose the reference

    3. Add index on the FK:
       ```prisma
       @@index([clock_logs_import_session_id], map: "idx_vpg_clock_logs_import_session_id")
       ```

    5. Run `npx prisma generate` to regenerate the Prisma Client.

    6. Verify: `npx tsc --noEmit` passes in src/backend/.
