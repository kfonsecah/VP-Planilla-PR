# Phase 63 Plan 04 - History Modal y Bulk Update

## Contexto
Permitir la visualización del historial de un parámetro legal específico mediante un modal. Además, incluir un modal exclusivo para actualizar masivamente los salarios mínimos según un nuevo decreto. Para cumplir con el control de acceso, los botones interactivos deben estar deshabilitados si el usuario es `payroll_manager` (`readOnly`).
