
**Modal renders:**
```tsx
{/* Add modal - triggered from employee header */}
<AddClockLogModal
  isOpen={isAddModalOpen}
  onClose={() => { setIsAddModalOpen(false); setSelectedEmployeeForAdd(null); }}
  employeeId={selectedEmployeeForAdd?.id}
  employeeName={selectedEmployeeForAdd?.name}
  onSuccess={() => { refetchEffectiveMarks(); }}
/>

{/* Edit modal - triggered from day row edit icon */}
<EditClockLogModal
  isOpen={!!selectedEntryForEdit}
  onClose={() => setSelectedEntryForEdit(null)}
  clockLog={selectedEntryForEdit!}
  onSuccess={() => { refetchEffectiveMarks(); }}
/>

{/* Void modal - triggered from edit context menu */}
<VoidClockLogModal
  isOpen={!!selectedEntryForVoid}
  onClose={() => setSelectedEntryForVoid(null)}
  clockLog={selectedEntryForVoid!}
  onSuccess={() => { refetchEffectiveMarks(); }}
/>
```

1. Employee header: Add "Agregar marca" button → calls `setSelectedEmployeeForAdd({id, name})` then `setIsAddModalOpen(true)`
2. Day card row: Edit icon → calls `setSelectedEntryForEdit(entry)` 
3. Day card row: Void via long-press or context menu → calls `setSelectedEntryForVoid(entry)`
